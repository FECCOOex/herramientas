Chat público

Espacio abierto para consultas breves (públicas) y orientación general.

Importante: este chat es público. Evita datos personales (DNI, teléfono, direcciones, datos de salud, etc.). Las respuestas no son inmediatas y dependen de disponibilidad. Si tu consulta es urgente o requiere privacidad, utiliza Consultas sindicales.
  • Consulta breve y con contexto (centro/etapa/jornada) sin identificar personas.
  • No se permite lenguaje ofensivo, difusión de datos personales ni contenidos discriminatorios.
  • Las respuestas orientan; para trámites o casos complejos se derivará a la vía de consulta formal.
Consejo: para agilizar, indica si tu consulta es sobre horario, permisos, jornada, sustituciones u otra temática.
